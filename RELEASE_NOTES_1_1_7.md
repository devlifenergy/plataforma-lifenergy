# Release 1.1.7 — Página institucional

## Alterações

- Nova página inicial institucional e responsiva.
- Botões `Entrar na Plataforma` direcionando para `/login`.
- Componente reutilizável `AppVersion`, com a versão lida diretamente do `package.json`.
- Tela de login atualizada para usar o componente `AppVersion`.
- Metadados institucionais e idioma da aplicação atualizados.
- Fontes locais adotadas para permitir builds sem dependências externas.
- Versão do projeto atualizada para `1.1.7`.

npm run build
