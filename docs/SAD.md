# Lifenergy Platform 2.0  
## Software Architecture Document - SAD v1.0

## 1. Princípio Central

O software será adaptado à Metodologia Lifenergy, e não o contrário.

## 2. Módulos Principais

- Lifenergy Platform
- Assessment Session
- Lifenergy Core Engine
- Report Engine
- Dashboard Engine

## 3. Assessment Session

A aplicação será organizada como uma sessão composta por etapas:

1. Registro
2. Identidade
3. Registro de Aplicação
4. Instruções
5. Transcrição do Fractal
6. Respostas
7. Hierarquia e Justificativas
8. Sentimento Final
9. Finalização

## 4. Decisão Metodológica

O entrevistado deve copiar o Fractal de Comportamento manualmente no campo indicado.  
Essa ação faz parte do processo metodológico e não será automatizada nesta fase.

## 5. Stack Técnica

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Vercel
